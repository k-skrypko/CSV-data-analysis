# Аналізатор CSV даних

Консольна програма для аналізу табличних даних у форматі CSV.  
Дозволяє завантажувати файли, виконувати статистичні операції, фільтрувати та сортувати дані, будувати графіки і зберігати результати.

---

## Автори

| Учасник              | Відповідальність                                                      |
| -------------------- | --------------------------------------------------------------------- |
| Учасник А (Христина) | Логіка аналізу: avg / min / max / filter / sort / top10               |
| Учасник B (Валерія)  | Валідація даних, збереження результатів, графіки, тести, документація |

---

## Структура проєкту

```
Парна практика/
├── main.py          — головний файл, запуск програми
├── validator.py     — перевірка CSV на коректність
├── saver.py         — збереження результатів у файл
├── visualizer.py    — побудова графіків
├── tests.py         — unit-тести
├── sales.csv        — дані про продажі товарів
├── students.csv     — дані про студентів
├── weather_2025.csv — погодні дані за 2025 рік
└── results/         — папка з результатами (створюється автоматично)
    └── charts/      — графіки у форматі PNG
```

---

## Вимоги

- Python 3.8 або новіший
- Бібліотеки: `pandas`, `matplotlib`

Встановити залежності:

```bash
pip install pandas matplotlib
```

---

## Як запустити

1. Завантажте або склонуйте репозиторій
2. Відкрийте термінал у папці проєкту
3. Запустіть програму:

```bash
python3 main.py
```

4. Введіть назву CSV файлу, наприклад:

```
sales.csv
```

5. Оберіть колонку і операцію

---

## Доступні операції

| Операція | Опис                                   | Приклад                       |
| -------- | -------------------------------------- | ----------------------------- |
| `avg`    | Середнє значення числової колонки      | Середня ціна товарів          |
| `min`    | Рядок з мінімальним значенням          | Найдешевший товар             |
| `max`    | Рядок з максимальним значенням         | Найдорожчий товар             |
| `filter` | Фільтрація за значенням                | Всі товари категорії "Одяг"   |
| `sort`   | Сортування за зростанням або спаданням | Товари від дешевих до дорогих |
| `top10`  | Топ-10 рядків за обраною колонкою      | 10 найпродаваніших товарів    |
| `exit`   | Завершення програми                    | —                             |

## Підтримувані CSV файли

### sales.csv — продажі товарів

| Колонка      | Тип   | Опис               |
| ------------ | ----- | ------------------ |
| Назва_товару | текст | Назва товару       |
| Категорія    | текст | Категорія товару   |
| Бренд        | текст | Виробник           |
| Ціна         | число | Ціна в гривнях     |
| Продано      | число | Кількість продажів |

### students.csv — студенти

| Колонка       | Тип   | Опис                 |
| ------------- | ----- | -------------------- |
| ПІБ           | текст | Прізвище та ім'я     |
| Спеціальність | текст | Напрям навчання      |
| Група         | текст | Навчальна група      |
| Середній_бал  | число | Середній бал (0–100) |

### weather_2025.csv — погода

| Колонка     | Тип   | Опис                  |
| ----------- | ----- | --------------------- |
| Дата        | текст | Дата спостереження    |
| Пора_року   | текст | Пора року             |
| Місяць      | текст | Назва місяця          |
| Температура | число | Температура в °C      |
| Опади       | число | Кількість опадів у мм |

## Валідація даних

Програма автоматично перевіряє завантажений файл на:

- порожні клітинки
- дублікати рядків
- некоректні типи даних
- порожній файл

---

## Графіки

Після завантаження файлу програма пропонує побудувати графік:

| Файл             | Тип графіку            | Опис                                 |
| ---------------- | ---------------------- | ------------------------------------ |
| sales.csv        | Горизонтальна діаграма | Топ-10 товарів за продажами          |
| students.csv     | Кругова діаграма       | Розподіл студентів по спеціальностях |
| weather_2025.csv | Лінійний графік        | Середня температура по місяцях       |

Графіки зберігаються в папку `results/charts/`

---

## Тестування

Запуск тестів:

```bash
python3 tests.py
```

Тести перевіряють:

- коректність підрахунку середнього значення
- коректність знаходження мінімуму та максимуму
- правильність роботи top-10
- валідацію порожнього DataFrame
- валідацію коректного DataFrame

---

## Збереження результатів

Після кожної операції програма пропонує зберегти результат у текстовий файл.  
Файли зберігаються в папку `results/` з назвою формату:

```
results/sales_avg_Ціна.txt
results/students_top10_Середній_бал.txt
```
